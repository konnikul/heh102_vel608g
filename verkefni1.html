Uploading my website to github

I started by installing Git on my computer. Then searched for a turtorial online by searching for "How to host a website on GitHub". I found this video wich explains the steps very well.

The steps are as follows

  • Make a new repository on GitHub
  • Right click on the folder where your files are stored and select "Git Bash Here". This will open a Git command module.
  • Type in the command "git init".
  • Type in the command "git add ." and press enter.
  • Type in the command "git commit -m "first commit"" and press enter
  • Type in the command "git remote ass originn url_to_your_repository" and press enter
  • Type in the command "git push -u origin main" and it will ask your GitHub username and password. Provide both and continue.

Now the code is uploaded to GitHub. I went to the settings for the repository I just created and found GitHub pages. Then I selected the main branch and choose the root folder. After saving these settings my website was published on GitHub.
